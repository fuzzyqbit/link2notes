Link To Notes

Drop an audio file, get beginner-friendly sheet music.

How to use

  1. Paste a YouTube link in the YouTube link box and hit Get MP3 →. Your link is copied to the clipboard and a converter site opens in a new tab.
  2. Paste the link into the converter, pick Audio / MP3, and download. If one site is broken or covered in ads, try another from the list.
  3. Drop the downloaded MP3 into the Audio file box.
  4. Pick your instrument and press Convert.

Important: only solo instrument tracks work well. One instrument playing one note at a time. No chords, no backing tracks, no singing over the music, no full bands. Pick a video that's just a single flute, violin, piano melody line, etc.

Trouble downloading? Other converters to try

Browsers can't pull audio directly from YouTube — only YouTube's own player can. Free converter sites come and go; if one stops working, try another:

  • savefrom.net — primary; also try the ss URL trick (replace youtube.com with ssyoutube.com in the video URL)
  • yt1s.com — fast, ad-heavy
  • ezmp3.cc — minimal UI
  • notube.net — straightforward
  • ddownr.com — supports music.youtube.com

Any format your browser can decode works (MP3, WAV, M4A, OGG, FLAC). Already have a recording? Skip the download step and drop the file in directly.
